# AiC Content Studio

AiC Content Studio is a Next.js 14 full-stack dashboard for AI-powered athlete video production.  
It uses Supabase for database, auth, and storage.

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Supabase
- Sonner toasts

## Setup

```bash
npm install
npm run dev
```

## Seed templates

```bash
npm run seed:templates
```

The seed script inserts 45 templates (15 categories x 3 variants) into the `templates` table.
